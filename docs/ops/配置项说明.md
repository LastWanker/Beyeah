# 配置项说明

## 后端环境变量
- `DB_HOST`：MySQL 主机，默认 `localhost`
- `DB_PORT`：MySQL 端口，默认 `3306`
- `DB_NAME`：数据库名，默认 `beyeah`
- `DB_USER`：数据库用户名，默认 `root`
- `DB_PASSWORD`：数据库密码，默认空
- `APP_AUTH_TOKEN_SECRET`：鉴权 token 签名密钥，生产环境必须替换
- `APP_AUTH_TOKEN_EXPIRE_SECONDS`：token 有效期（秒），默认 `259200`（3 天）
- `APP_ADMIN_AUTH_TOKEN_SECRET`：管理员鉴权 token 签名密钥，默认 `${APP_AUTH_TOKEN_SECRET}-admin`
- `APP_ADMIN_AUTH_TOKEN_EXPIRE_SECONDS`：管理员 token 有效期（秒），默认 `43200`（12 小时）
- `APP_AUTH_LOGIN_MAX_ATTEMPTS`：登录失败最大次数，默认 `5`
- `APP_AUTH_LOGIN_WINDOW_SECONDS`：失败统计窗口（秒），默认 `600`
- `APP_AUTH_LOGIN_LOCK_SECONDS`：触发限制后的锁定时长（秒），默认 `900`
- `APP_CORS_ALLOWED_ORIGINS`：允许跨域来源，逗号分隔

## 默认读取位置
- `backend/src/main/resources/application.properties`

## 本地开发建议
1. 复制根目录 `.env.example` 为 `.env`。  
2. 在 IDE 运行配置中加载 `.env`。  
3. 至少保证 `DB_PASSWORD` 与 `APP_AUTH_TOKEN_SECRET` 已配置。

## 第三阶段批次 A 新增配置
- `APP_ORDER_AUTO_CLOSE_SECONDS`：未支付订单自动关闭阈值（秒），默认 `1800`（30 分钟）
- `APP_ORDER_AUTO_CLOSE_BATCH_SIZE`：每次扫描并处理的最大订单数，默认 `100`
- `APP_ORDER_AUTO_CLOSE_FIXED_DELAY_MS`：定时任务固定间隔（毫秒），默认 `60000`
